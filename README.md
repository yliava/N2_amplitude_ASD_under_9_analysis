# Выделение N2 из аудиальных вызванных потенциалов и оценка её изменения в зависимости от интенсивности звука

Этот репозиторий содержит код и результаты анализа аудиальных вызванных потенциалов (EAP) с фокусом на компоненте **N2** у детей с РАС младше 9 лет.  
Основная цель работы — **выделить компоненту N2** из предобработанных данных ЭЭГ и **количественно описать её изменение** в зависимости от уровня интенсивности звука (дБ).
Для описания зависимости амплитуды N2 от интенсивности звука используются **медианные локальные регрессии**:

- для каждого участника собираются пары:  
  `интенсивность звука (дБ)` → `амплитуда N2`,
- с помощью `LinearRegression` оценивается **наклон** (slope) этой зависимости на локальных участках - пары условий,
- для каждого участника берётся **медиана** его локальных значений и рассчитывается агрегированный показатель **ME**.

## Структура репозитория

- `notebooks/`  
  - `N2_amplitude_ASD_under_9_analysis.ipynb` — основной Jupyter-ноутбук с полным рабочим пайплайном:
    - загрузка данных,
    - выделение N2,
    - расчёт показателя ME,
    - визуализация,
    - сохранение итоговых таблиц.

- `figures/`  
  - Готовые изображения, иллюстрирующие распределение медианных значений ME(N2) по:
    - участникам (ID),
    - уровням интенсивности звука (например, 50 и 70 дБ),
    - группам/подвыборкам (при необходимости).

---

## Используемые данные

- Предобработанные ERP-данные для детей с РАС возрастом **до 9 лет**.
- Для каждого участника доступны усреднённые EAP по условиям, отличающимся **уровнем интенсивности звука**.
- Анализ строится на отдельных файлах/записях формата EDF (один файл на участника), которые загружаются с диска с помощью `mne`.

> В ноутбуке путь к данным задаётся явно (через `cd YOURPATH` и переменные типа `data_path_ASD`), их необходимо заменить на свои реальные пути.

## Как использовать ноутбук

1. **Клонировать репозиторий** или скачать его архивом.  
2. Установить все необходимые библиотеки из `requirements.txt`
3. Открыть `notebooks/N2_amplitude_ASD_under_9_analysis.ipynb` в Jupyter Lab / Jupyter Notebook / VS Code.  
4. Обновить пути:
   - рабочий каталог (`cd YOURPATH`),
   - путь к данным (`data_path_ASD` и т.п.),
   - путь для сохранения результатов (таблиц и рисунков).
5. Запускать ячейки по порядку.
